# RUNBOOK

## Use When
the repo contains templates, generators, flattened outputs, cached derivatives, exporters, or plugin-driven generated artifacts and the user needs to know whether to edit source or generated output.

## Workflow
1. Identify canonical source-of-truth.\n2. Identify all transforms and derived outputs.\n3. Distinguish source edits from generated artifact regeneration.\n4. Identify drift risk if only generated outputs change.\n5. Feed the result into impact analysis or implementation guidance.
